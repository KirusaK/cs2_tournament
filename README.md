# CS2 Tournament Manager (Educational Project)

A web application designed to manage CS2 esports tournaments. This project was developed for educational purposes to practice Fullstack development skills using Node.js, PostgreSQL, and modern Frontend techniques.

## 🚀 Key Features

- **Tournament Brackets:** Visual representation of match progress.
- **Team Management:** Database storage for teams and player profiles.
- **Match Status:** Real-time tracking of scores and scheduled games.

## 🛠 Tech Stack

- **Backend:** Node.js, Express.js.
- **Database:** PostgreSQL (via `pg` library).
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (Fetch API).
- **Security:** Dotenv for environment variable management.

## 📦 Getting Started

1. **Database:** Create a PostgreSQL database and execute the script found in `/db/schema.sql`.
2. **Configuration:** Create a `.env` file in the `/backend` folder and provide your credentials (DB_USER, DB_PASSWORD, etc.).
3. **Install Dependencies:** Run `npm install` inside the `backend` directory.
4. **Start the Server:** Run `node index.js`.
5. **Frontend:** Open `index.html` in your web browser.

# CS2 Tournament Manager (Учебный проект)

Это веб-приложение для управления киберспортивными турнирами по CS2. Проект создан в учебных целях для отработки навыков Fullstack-разработки (Node.js + PostgreSQL + Frontend).

## 🚀 Основной функционал

- **Турнирная сетка (Brackets):** Визуальное отображение матчей.
- **Управление командами:** Хранение данных о командах и игроках в базе данных.
- **Статус матчей:** Отслеживание текущих результатов и завершенных игр.

## 🛠 Технологический стек

- **Backend:** Node.js, Express.js.
- **Database:** PostgreSQL (используется библиотека `pg`).
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (Fetch API).
- **Environment:** Dotenv для защиты данных.

## 📦 Как запустить проект

1. **База данных:** Создайте базу в PostgreSQL и выполните скрипт из папки `/db/schema.sql`.
2. **Конфигурация:** Создайте файл `.env` в папке `/backend` и укажите свои данные (DB_USER, DB_PASSWORD и т.д.).
3. **Установка зависимостей:** В папке `backend` выполните `npm install`.
4. **Запуск сервера:** Выполните команду `node index.js`.
5. **Фронтенд:** Откройте `index.html` в браузере.
